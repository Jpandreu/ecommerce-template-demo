## Troubleshooting 🔧

### Common Issues
**Problem**: Pages won't load
**Solution**: Check file permissions (644 for files, 755 for folders)

**Problem**: PayPal not working
**Solution**: Verify Client ID in checkout.html and use sandbox mode for testing

**Problem**: Images not showing
**Solution**: Ensure images folder uploaded correctly and paths are relative

**Problem**: Mobile layout issues
**Solution**: Clear browser cache and test on actual mobile devices
